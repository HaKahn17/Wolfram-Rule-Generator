# Wolfram-Rule-Generator

This code creates cellular automaton based on their rule number as described here: https://writings.stephenwolfram.com/2017/06/oh-my-gosh-its-covered-in-rule-30s/

There are global variables that change the height, width, color, pattern, and filename of the resulting image. Additionally, the triangle option toggles both random start conditions and the shape of the image (random start, square image vs single pixel start, triangle image).
